# Runtime dependencies for the Neural Chess Engine & Lichess Bot.
#
# The engine itself needs torch / python-chess / numpy; the lichess-bot
# framework in ./lichess-bot needs PyYAML / requests / backoff / rich.
#
# On the Ampere ARM (aarch64) Oracle shape, `pip install torch` pulls the
# correct CPU wheel automatically. On an x86 host, install torch from the
# CPU index instead to avoid the large CUDA build:
#   pip install torch --index-url https://download.pytorch.org/whl/cpu

# --- engine ---
torch
python-chess>=1.11
numpy

# --- lichess-bot framework ---
PyYAML
requests
backoff
